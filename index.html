Foundation

Claude Code vs ChatGPT

Both can write excellent VBA macros. Here's how they differ for PowerPoint automation work, so you can choose the right tool for each task.

ChatGPT

  • Best for: Quick one-off macros, conversational iteration
  • How to use: Web app (chat.openai.com) or desktop app
  • Strengths: Great at explaining code, beginner-friendly tone
  • VBA quality: Very good, occasionally uses deprecated patterns
  • Context: Remembers conversation history for follow-ups
  • Tip: Use GPT-4o for best VBA output. Start with the session prompt from this guide.

Claude Code

  • Best for: Complex multi-file operations, batch processing, production code
  • How to use: Terminal CLI (claude), also web (claude.ai)
  • Strengths: Superior error handling, longer outputs, reads your files directly
  • VBA quality: Excellent, consistent modern patterns, thorough edge cases
  • Context: Can read your .pptm files directly in CLI mode
  • Tip: Claude Code CLI can read your VBE modules and suggest fixes in-place
💡

Recommendation: Start with whichever you already use. The prompts in this guide work identically in both — we've tested and adapted them for each. Toggle between versions using the switch above. If you're curious about Claude Code, the prompts here are a great way to try it.

🎓 Object Hierarchy

Essential

Every VBA macro navigates this tree. Understand it and you can describe any task to Claude:

Hierarchy
Application                     ' PowerPoint itself
  └── Presentations               ' All open files
       └── Presentation            ' One .pptx/.pptm
            ├── Slides              ' Collection of slides
            │    └── Slide           ' Single slide
            │         ├── Shapes   ' All objects on slide
            │         │    ├── TextFrame → TextRange
            │         │    ├── Fill / Line
            │         │    ├── Table → Cell → Shape
            │         │    ├── Chart
            │         │    ├── GroupItems ' nested shapes
            │         │    └── PictureFormat
            │         └── SlideShowTransition
            ├── SlideMaster
            │    └── CustomLayouts
            ├── Designs             ' Themes
            └── Tags                ' Custom metadata

💬 How to Describe Macros to Claude

The better your prompt, the better the macro. Structure your requests with:

  1. What — the specific action ("resize all images")
  2. Where — scope (all slides? current slide? selected shapes?)
  3. Rules — constraints ("only images wider than 800px", "skip grouped shapes")
  4. Output — expected result ("center on slide, maintain aspect ratio")
  5. Report — what to show when done ("count of images processed")
Claude Code Prompt
Write a VBA macro for PowerPoint that: - Goes through ALL slides in the active presentation - Finds every image/picture shape - Resizes each to exactly 800pt wide, maintaining aspect ratio - Centers each image horizontally on the slide - Skips any image inside a group - Shows a message box with count of images processed when done

🔄 The Iteration Loop

Describe → Claude writes macro

Be specific about your presentation structure and needs.

Paste into VBE → Run on test file

Always test on a COPY first. Never untested macros on live client files.

Check results → Report issues back

Tell Claude exactly what went wrong: error messages, wrong shapes affected.

Claude fixes → Re-test → Ship

Save working macros in your personal library module for reuse.

⚠️

Golden rule: Always Ctrl+S before running any new macro. VBA has no undo.

📄 Duplicate Slide with New Layout

Essential

Duplicates the current slide and applies a different layout by name.

VBA
Sub DuplicateWithLayout(layoutName As String)
    On Error GoTo ErrHandler
    Dim sld As Slide, newSld As Slide
    Dim lo As CustomLayout

    Set sld = ActiveWindow.View.Slide
    sld.Duplicate
    Set newSld = ActivePresentation.Slides(sld.SlideIndex + 1)

    For Each lo In ActivePresentation.SlideMaster.CustomLayouts
        If lo.Name = layoutName Then
            newSld.CustomLayout = lo
            Exit For
        End If
    Next lo

    MsgBox "Duplicated with layout: " & layoutName
    Exit Sub
ErrHandler:
    MsgBox "Error: " & Err.Description, vbExclamation
End Sub

📑 Auto-Generate Table of Contents

Agency

Reads all slide titles and generates a TOC slide at position 2.

VBA
Sub GenerateTOC()
    On Error GoTo ErrHandler
    Dim pres As Presentation, sld As Slide
    Dim tocSlide As Slide
    Dim tocText As String, slideTitle As String
    Dim itemNum As Long

    Set pres = ActivePresentation

    For Each sld In pres.Slides
        If sld.SlideIndex > 1 Then
            On Error Resume Next
            slideTitle = sld.Shapes.Title.TextFrame.TextRange.Text
            On Error GoTo ErrHandler
            If Len(slideTitle) > 0 Then
                itemNum = itemNum + 1
                tocText = tocText & itemNum & ".  " & slideTitle & vbCrLf
            End If
            slideTitle = ""
        End If
    Next sld

    Set tocSlide = pres.Slides.Add(2, ppLayoutText)
    tocSlide.Shapes.Title.TextFrame.TextRange.Text = "Table of Contents"
    tocSlide.Shapes(2).TextFrame.TextRange.Text = tocText
    tocSlide.Shapes(2).TextFrame.TextRange.Font.Size = 14

    MsgBox "TOC created with " & itemNum & " items."
    Exit Sub
ErrHandler:
    MsgBox "Error: " & Err.Description, vbExclamation
End Sub

🎨 Global Font Replacement

Essential

Replaces ALL instances of one font with another, including inside tables, groups, and charts.

VBA
Sub GlobalFontReplace(oldFont As String, newFont As String)
    On Error GoTo ErrHandler
    Dim sld As Slide, count As Long

    For Each sld In ActivePresentation.Slides
        count = count + SwapFontInShapes(sld.Shapes, oldFont, newFont)
    Next sld

    MsgBox "Replaced '" & oldFont & "' → '" & newFont & "' in " & count & " text runs."
    Exit Sub
ErrHandler:
    MsgBox "Error: " & Err.Description, vbExclamation
End Sub

Private Function SwapFontInShapes(shps As Shapes, oF As String, nF As String) As Long
    Dim shp As Shape, c As Long, run As TextRange
    For Each shp In shps
        If shp.HasTextFrame Then
            For Each run In shp.TextFrame.TextRange.Runs
                If run.Font.Name = oF Then
                    run.Font.Name = nF: c = c + 1
                End If
            Next run
        End If
        If shp.HasTable Then
            Dim r As Long, cl As Long
            For r = 1 To shp.Table.Rows.Count
                For cl = 1 To shp.Table.Columns.Count
                    Dim cellRun As TextRange
                    For Each cellRun In shp.Table.Cell(r, cl).Shape.TextFrame.TextRange.Runs
                        If cellRun.Font.Name = oF Then
                            cellRun.Font.Name = nF: c = c + 1
                        End If
                    Next
                Next cl
            Next r
        End If
        If shp.Type = msoGroup Then
            Dim gs As Shape
            For Each gs In shp.GroupItems
                If gs.HasTextFrame Then
                    For Each run In gs.TextFrame.TextRange.Runs
                        If run.Font.Name = oF Then
                            run.Font.Name = nF: c = c + 1
                        End If
                    Next
                End If
            Next gs
        End If
    Next shp
    SwapFontInShapes = c
End Function

Troubleshooting

Common VBA errors and how to fix them.

🚨 Runtime Error 91: Object variable not set

Cause: Accessing a shape/slide that doesn't exist.

Fix: Check with If Not shp Is Nothing Then or verify .Count before indexing.

🚨 "The specified value is out of range"

Cause: Invalid index after deletion mid-loop.

Fix: Loop backwards: For i = .Count To 1 Step -1

🚨 "Method or data member not found"

Cause: Accessing .TextFrame on a shape without one (like a picture).

Fix: Always check If shp.HasTextFrame Then first.

🚨 Macros disappear after saving as .pptx

Cause: .pptx strips all VBA code.

Fix: Save As → .pptm (Macro-Enabled Presentation).

🚨 Macro skips shapes inside groups

Cause: Slide.Shapes doesn't include items inside groups.

Fix: Check shp.Type = msoGroup and recurse through shp.GroupItems.

🚨 "Automation Error" on Save

New

Cause: File locked by another process (OneDrive sync, antivirus).

Fix: Pause OneDrive sync, save to a local folder, or try SaveCopyAs instead of Save.

💡

Debugging tip: Use Ctrl+G in VBE for the Immediate Window. Type ? ActivePresentation.Slides(1).Shapes.Count to test expressions live without running a full macro.
